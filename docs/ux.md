# QQ Group Bot — UX Templates

All templates use `{variable}` placeholders for string interpolation.
Tone: friendly 群友 for chat; strict but respectful for moderation.

---

## 1. /help — Full Text

```
欢迎使用群机器人！以下是所有可用指令：

【聊天 & 模仿】
/mimic @群友 [话题]   — 让我模仿某位群友的说话风格回复一句
/mimic_on @群友       — 开启持续模仿模式（全群生效，任何人可关闭）
/mimic_off            — 关闭当前的模仿模式

【群规 & 管理】（仅管理员）
/rule_add <描述>       — 添加一条群规或违规示例
/rule_false_positive <消息ID> — 标记一条 AI 误判，撤销对应处罚

【申诉】
/appeal               — 申诉你最近一次受到的处罚（处罚后24小时内有效）

【查看信息】
/rules                — 查看本群当前所有群规
/stats                — 查看本群近7天的统计数据

如有疑问请联系群管理员。
```

## 2. 各指令回复模板

### /mimic @user [topic]

**成功：**
```
{generated_reply}
```

**错误 — 无历史记录（E002）：**
```
@{nickname} 在本群没有历史消息记录，无法进行模仿。
```

**警告 — 历史太少（E003，仍继续执行）：**
```
⚠️ @{nickname} 目前只有 {count} 条历史消息，模仿效果可能不准确。

{generated_reply}
```

**错误 — 模仿自己（E017）：**
```
我没办法模仿我自己啦。
```

---

### /mimic_on @user

**成功：**
```
模仿模式已开启：本群之后的回复将模仿 @{nickname} 的说话风格。使用 /mimic_off 可随时关闭。
```

**错误 — 已有模仿会话活跃（E013）：**
```
当前已在模仿 @{active_nickname}，请先发送 /mimic_off 关闭后再试。
```

**错误 — 目标无历史（E002）：**
```
@{nickname} 在本群没有历史消息记录，无法开启模仿模式。
```

---

### /mimic_off

**成功：**
```
模仿模式已关闭，恢复正常聊天模式。
```

**无活跃会话：**
```
当前没有开启模仿模式，无需关闭。
```

---

### /rule_add <description>

**成功：**
```
规则已添加（ID: {rule_id}）：{first_80_chars}...
```

**错误 — 无权限（E001）：**
```
没有权限。只有管理员可以添加规则。
```

**错误 — 描述为空：**
```
规则描述不能为空，请重新输入。
```

**错误 — 规则过长（E014）：**
```
规则内容太长（最多2000字），请精简后重试。
```

**警告 — 疑似重复（E015）：**
```
已存在高度相似的规则（ID: {existing_id}）。如需强制添加，请使用：
/rule_add --force {description}
```

**错误 — 写入失败（E011）：**
```
规则保存失败，请稍后再试。
```

---

### /rule_false_positive <msg_id>

**成功：**
```
消息 {msg_id} 已标记为误判，处罚已撤销，已将该消息作为反例加入学习库。
```

**错误 — 无权限（E001）：**
```
没有权限。只有管理员可以操作此指令。
```

**错误 — 记录不存在（E007）：**
```
未找到消息 {msg_id} 的审核记录，请确认 ID 是否正确。
```

**错误 — 已撤销（E008）：**
```
该处罚已经被撤销过了，无需重复操作。
```

**特殊 — 目标是踢出且已执行：**
```
数据库记录已更正，但用户已被移出群聊，无法自动恢复，请手动重新邀请。
```

---

### /appeal

**成功提交：**
```
@{user} 申诉已提交，禁言已暂时解除，正在等待管理员审核。
```

**错误 — 无处罚记录：**
```
未找到你的近期处罚记录，无法发起申诉。
```

**错误 — 超过24小时（E005）：**
```
申诉窗口已关闭（仅限处罚后24小时内）。如有异议请直接联系管理员。
```

**错误 — 已申诉（E006）：**
```
你已经对这次处罚提交过申诉了，请等待管理员审核结果。
```

---

### /rules

**有规则时：**
```
本群当前群规（共 {total} 条，显示第 {page_start}–{page_end} 条）：

{rule_number}. {rule_text}
...

如需查看更多，发送 /rules page {next_page}
```

**无规则：**
```
本群尚未配置任何群规。管理员可使用 /rule_add 添加。
```

---

### /stats

**正常：**
```
本群近7天统计数据：
- 处理消息数：{messages_processed}
- 检测违规数：{violations_detected}
- 已执行处罚：{punishments_issued}
- 申诉记录：{appeals_total}（{appeals_approved} 条获批）
- 今日剩余处罚配额：{remaining}/{cap}
- 当前模仿模式：{OFF | 正在模仿 @{nickname}}
```

**查询超时（E011）：**
```
统计数据暂时无法获取，请稍后再试。
```

## 3. 审核通知模板

### severity 1 — 删除 + 警告

```
@{user} 你的消息因违反群规已被删除，请注意言行（第 {warn_count} 次警告）。
原因：{reason}
```

### severity 2 — 删除 + 警告（累计提示）

```
@{user} 违规警告（第 {warn_count} 次）。
原因：{reason}
累计3次警告将自动触发禁言，请遵守群规。
```

### severity 3 — 删除 + 禁言10分钟

```
@{user} 因违规已禁言10分钟。
原因：{reason}
如认为处罚有误，可在24小时内发送 /appeal 申诉。
```

### severity 4 — 删除 + 禁言1小时

```
@{user} 因严重违规已禁言1小时。
原因：{reason}
如认为处罚有误，可在24小时内发送 /appeal 申诉。
```

### severity 5 — 删除 + 踢出

```
用户 {nickname}（{user_id}）因严重违规已被移出群聊。
原因：{reason}
```

## 4. 申诉流程 UX

### 用户发起申诉

用户在群内发送 `/appeal`。

**Bot 回复（成功）：**
```
@{user} 申诉已提交，禁言已暂时解除，等待管理员审核。
```

**同时私信通知管理员（或群内通知）：**
```
[申诉通知] @{user} 对以下处罚提出申诉：
- 处罚类型：{punishment_type}
- 处罚时间：{punishment_time}
- AI 判定原因：{ai_reason}
- 原始消息 ID：{msg_id}

请使用 /rule_false_positive {msg_id} 批准申诉，或（将来）使用 /appeal_deny {msg_id} 拒绝。
```

### 管理员批准（/rule_false_positive）

```
@{user} 申诉成功，处罚已撤销。感谢你的反馈，我们已将此情况记入学习库。
```

### 管理员拒绝（/appeal_deny，第二阶段功能）

```
@{user} 申诉未获批准，原处罚维持有效。如有疑问请联系管理员。
```

## 5. 模仿回复格式

模仿回复直接输出生成的文本，不添加任何前缀。`[模仿 @{nickname}]` 标记仅保留在服务端日志中（pino info 级别），不出现在群内消息里。

## 6. 错误码对应用户消息

| 错误码 | 名称 | 用户可见文案 |
|--------|------|-------------|
| E001 | PERMISSION_DENIED | 没有权限，此操作仅限管理员。 |
| E002 | USER_NOT_FOUND | 未找到 @{user} 的历史记录，无法执行。 |
| E003 | INSUFFICIENT_HISTORY | ⚠️ @{user} 历史消息仅 {count} 条，结果可能不准确。 |
| E004 | DAILY_CAP_REACHED | （仅通知管理员）⚠️ 今日自动处罚已达上限，后续违规将仅记录，不自动处罚，请管理员及时处理。 |
| E005 | APPEAL_EXPIRED | 申诉窗口已过期（仅限24小时内），请直接联系管理员。 |
| E006 | APPEAL_DUPLICATE | 你已对此处罚提交过申诉，请耐心等待审核结果。 |
| E007 | NO_PUNISHMENT_RECORD | 未找到消息 {msg_id} 的审核记录，请确认 ID。 |
| E008 | ALREADY_REVERSED | 该处罚已经撤销过了，无需重复操作。 |
| E009 | CLAUDE_API_ERROR | （内部，无用户消息）AI 服务暂时不可用，本次审核已跳过，请稍后重试。 |
| E010 | CLAUDE_PARSE_ERROR | （内部，无用户消息）AI 返回格式异常，已按无违规处理，已记录日志。 |
| E011 | DB_ERROR | 服务暂时不可用，请稍后再试。 |
| E012 | NAPCAT_ACTION_FAIL | （群内通知管理员）执行操作失败（{action}），请管理员手动处理，详情已记录日志。 |
| E013 | MIMIC_SESSION_ACTIVE | 当前已在模仿 @{active_user}，请先发送 /mimic_off 关闭后再试。 |
| E014 | RULE_TOO_LONG | 规则内容过长（最多2000字），请精简后重试。 |
| E015 | RULE_DUPLICATE | 已存在相似规则（ID: {id}），如需强制添加请使用 --force 参数。 |
| E016 | WHITELIST_MEMBER | （静默，仅记录日志，不通知用户） |
| E017 | SELF_MIMIC | 我没办法模仿我自己啦。 |

## 7. 速率限制触发消息

### 用户触发个人限速

```
你的操作太频繁了，请等待 {cooldown} 秒后再试。
```

### /mimic 限速（每分钟3次）

```
/mimic 使用次数超限，请 {cooldown} 秒后再试。
```

### 群级 bot 回复限速（静默停止）

不发送任何消息，静默停止回复，日志记录即可。

## 8. 降级 / 故障安全消息

### Claude API 不可用（E009）

不向群内发送任何消息。日志记录错误。审核结果默认为无违规，处罚跳过。

### Claude 返回格式异常（E010）

不向群内发送任何消息。日志记录，按无违规处理。

### OneBot 动作执行失败（E012）

```
（管理员注意）执行 {action} 操作失败，请手动处理用户 {user_id}。错误详情已记录日志。
```

### 机器人初始化中（/help 在启动期间）

```
机器人正在启动中，请稍候片刻再试。
```

### /help 在私聊中发送

```
本机器人仅在群聊中提供服务，请在群里使用相关指令。
```

### 数据库暂时不可用（E011）

```
服务暂时不可用，请稍后再试。
```
